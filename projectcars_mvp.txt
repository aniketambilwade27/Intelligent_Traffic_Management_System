## ğŸš€ Quick Start

### Backend Setup

1. **Create project structure:**
```bash
mkdir -p projectcars-mvp/backend/app
mkdir -p projectcars-mvp/backend/uploads
mkdir -p projectcars-mvp/backend/processed
cd projectcars-mvp/backend
```

2. **Create virtual environment:**
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. **Install dependencies:**
Create `requirements.txt` and install

4. **Run backend:**
```bash
uvicorn app.main:app --reload --port 8000
```

### Frontend Setup

1. **Create React app:**
```bash
cd projectcars-mvp
npm create vite@latest frontend -- --template react
cd frontend
```

2. **Install dependencies:**
```bash
npm install
npm install -D tailwindcss postcss autoprefixer
npm install axios react-router-dom react-query
```

3. **Setup Tailwind:**
```bash
npx tailwindcss init -p
```

4. **Run frontend:**
```bash
npm run dev
```

## ğŸ¯ API Endpoints

- `POST /api/videos/upload` - Upload video
- `GET /api/videos` - List all videos
- `GET /api/videos/{id}` - Get video details
- `POST /api/videos/{id}/process` - Start processing
- `GET /api/videos/{id}/status` - Get processing status
- `GET /api/videos/{id}/download` - Download processed video
- `DELETE /api/videos/{id}` - Delete video
- `GET /api/analytics/summary` - Get analytics summary

# Contributing to Traff-AI

Thanks for your interest in contributing! Please follow these steps to get started.

1. Fork the repo and clone your fork locally.
2. Create a feature branch:

```bash
git checkout -b feat/short-description
```

3. Make small, focused commits with clear messages.

4. Write tests for any new features or bug fixes.

5. Run linters and formatters (if configured).

6. Push your branch and open a pull request describing the change.

Code review
- PRs will be reviewed and may request changes before merging.

Reporting issues
- Use the issue tracker to report bugs or request features. Provide steps to reproduce and logs if possible.
